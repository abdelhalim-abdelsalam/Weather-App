## Project setup
1. Install node dependencies
```
npm install
```
2. Create .env file in root folder and add this
```
VUE_APP_API_KEY={OpenWeather api token}
```
3. Compile for development mode

npm run dev
```
4. Compile for production mode
```
npm run build
```
